Step 1: Copy and paste each signature code and place it in the left box found in this URL: https://www.w3schools.com/html/tryit.asp?filename=tryhtml_links_w3schools Step 2: Confirm the email signature produced on the right side is correct. Step 3: Copy the produced signature (on the right side). Step 4: Open the Gmail account. In the top right, click Settings. Scrool down the first setting pages, and in the "Signature" section, create a new signature, and name it. Paste the email signature text in the box. Set the new signature as a "Default Signature". At the bottom of the page, click Save Changes. You must do this to save the signature.
